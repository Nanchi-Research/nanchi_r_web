Getting Started

GitHub Setup

Cloning Repositories

  1. Install Git
  2. Open terminal/command prompt
  3. Run: git clone https://github.com/Nanchi-Research/[repository-name].git
  4. Navigate to project folder: cd [repository-name]

Basic Git Commands

  • git pull - Get latest changes
  • git status - Check current changes
  • git add . - Stage all changes
  • git commit -m "Your message" - Commit changes
  • git push - Push to remote repository

Python Setup

Creating Virtual Environment

  1. Install Python 3.8+ from python.org
  2. Create virtual environment: python -m venv venv
  3. Activate environment:
    • Windows: venv\Scripts\activate
    • Mac/Linux: source venv/bin/activate

Installing Requirements

  1. Navigate to project folder
  2. Install dependencies: pip install -r requirements.txt
  3. Verify installation: python -c "import torch; print(torch.__version__)"

Project Structure

Typical Layout

project-root/
├── data/            # Dataset files
├── docs/            # Documentation
├── models/          # Pretrained models
├── notebooks/       # Jupyter notebooks
├── src/             # Source code
│   ├── utils/       # Utility functions
│   ├── training/    # Training scripts
│   └── evaluation/  # Evaluation scripts
├── requirements.txt # Python dependencies
└── README.md        # Project overview

Important Files

  • README.md - Project overview and setup
  • requirements.txt - Python dependencies
  • config.yaml - Configuration parameters
  • train.py - Main training script

Running Projects

Basic Execution

  1. Activate your virtual environment
  2. Navigate to project root
  3. Run main script: python train.py --config configs/default.yaml
  4. Monitor training (TensorBoard): tensorboard --logdir runs/

Common Commands

  • python train.py --help - Show all options
  • python evaluate.py --model checkpoint.pt - Run evaluation
  • jupyter notebook - Launch Jupyter notebook server
  • pytest tests/ - Run unit tests
